Willert Software Tools RXF - Introduction

Embedded UML RXF

Embedded UML RXF™ (Real-time eXecution Framework) from Willert Software Tools is the interface between a UML model and the target platform. It facilitates the use of UML in connection with code generation for even small target platforms.

Using These Help Files

All major pages of the help contents are available via the navigation table-of-contents displayed on the left side of each page. Under the "Support" section you can also open an extended "Table of Contents" or an "Alphabetical Index" containing keywords used inside the help pages and the available links to the matching help sections. If you experience problems with the left side navigation table-of-contents, please make sure JavaScript is enabled and allowed for these help files in your browser.

How to Get Started

It is recommended to get familiar with the installation of the RXF by reading "Usage" section of this help. At least you should take the time to read the "Installation" section and then go through the "Getting Started" tutorial.
