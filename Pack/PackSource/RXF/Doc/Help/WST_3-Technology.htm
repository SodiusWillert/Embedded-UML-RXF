RXF Technology

This chapter describes the more detailed working of the Real-time eXecution Framework and allows to fine-tune applications using the RXF to improve and optimize them and the co-working with our RXF technologies.

It is explained

  • how concurrency is implemented and how to model to maximize the benefits of your RTOS
  • how error handling is implemented and how you can insert your own error handling
  • what packages and profiles come with the product
