Profiles


The product @WST_PRODUCT_NAME@ comes with three profiles, which you need to use when creating a model:
  • RXF with a few stereotypes and package used in all RXF products
  • RXFMonitor with stereotypes to use the UML Target Debugger, which uses a monitor as part of the RXF
  • RXFStereotypesCpp with stereotypes you can use when modeling based on the RXF
